# 🚦 AI智慧交通运输分析与决策平台

## 📖 项目简介

本项目基于深圳交通开放数据，构建 AI 智慧交通运输分析与决策平台。

通过 Python 完成交通数据清洗、统计分析、交通运行监测以及可视化展示，实现交通运行状态感知、拥堵识别与辅助决策功能。

---

## 🎯 项目目标

- 交通运行状态监测
- 交通拥堵分析
- 交通指数统计
- 车速趋势分析
- AI辅助决策建议
- 智慧交通可视化指挥大屏

## 📊 功能模块

### 1. KPI实时监测

展示：

- 平均车速
- 平均交通指数
- 街道数量
- 数据总量

---

### 2. 交通运行分析

实现：

- 车速分布分析
- 交通指数分析
- 交通状态识别

---

### 3. Top10拥堵街道

识别：

- 最拥堵街道
- 平均交通指数排序
- 重点监控区域

---

### 4. 车速趋势分析

分析：

- 时间变化趋势
- 交通运行规律
- 高峰时段特征

---

### 5. AI辅助决策

根据交通指数自动生成：

- 畅通状态建议
- 缓行状态建议
- 拥堵状态建议
